Understanding HTML

  • HTML acts like building blocks for the internet. These blocks have tags, like special badges called attributes.
  • Attributes give extra information and are always introduced at the beginning of the tag.
  • Imagine a link on a webpage. The href attribute in the link's tag is like a magical address book, guiding the computer to the linked page.
  • Pictures on a webpage, known as images, have a src attribute that's like a treasure map showing where the picture is hidden.
  • The width and height attributes of a picture indicate its size, and the alt attribute is like a short description in case the picture can't be seen.
  • Let's talk about style â€“ the look and feel of a webpage. The style attribute is like a box of crayons, allowing you to choose colors, fonts, and sizes to make things visually appealing.
  • Remember the lang attribute? It's like saying, "Hey, computer, this webpage speaks a particular language!"
  • The lang attribute specifies the language of the whole webpage. Finally, the title attribute gives a bit more info about something on the webpage.
  • Think of a website as a storybook. The big titles at the beginning of a section are like chapter names - they're called headings.
  • Headings have different levels, like Chapter 1 and Chapter 2. In website language, we use <h1> for the main heading and <h2> for smaller ones.
  • The regular text on a website, like paragraphs, is created using the <p> tag. It helps to keep things neat and organized, just like paragraphs in a book. So, headings are like big chapter titles, and paragraphs are like the regular text in a story. Easy, right?
