INTRODUCTION TO WEB DEVELOPMENT

"Web developer" is the umbrella term that is used to encompass the wide variety of skill sets you might have in web development.
They are responsible for creating and maintaining websites.


THERE ARE THREE MAIN TYPES OF WEB DEVELOPERS:

Front-end developer
A front end developer works on the client side and they design the user interfaces that we see on
the screen. These are the text and beautiful colors. They primarily use HTML, CSS, and Javascript.
Taking this course implies
that you're a front end developer!


Back-end developer
A back end developer works on the server side to ensure that everything in the system is running smoothly.
Backend developers use many languages. However, the primary ones are Python, PHP, Ruby, and Java.


Full-stack developer
Full-stack developers are proficient in both backend and frontend.

Others may include Web security developers, Wordpress developers, and Mobile web developers

INTRODUCTION TO HTML

HTML stands for Hyper Text Markup Language

It is a markup language, not a programming language.

WHY?

Simple...because it uses something called markups also known as tags

Tags act as labels that store small pieces of information that get displayed on the browser.

A starting and ending tag form an element (<>)

All HTML elements use tags. HTML elements are the text-building blocks of all HTML pages.

These elements are written in a program called a text editor

Examples are Visual Studio Code, Notepad, Notepad++ and Sublime Text Editor
