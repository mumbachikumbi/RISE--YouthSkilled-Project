INTRODUCTION TO CSS

CSS stands for Cascading Style Sheets

CSS is like a set of instructions that tells the computer how to arrange and show things on a webpage made with HTML.

CSS is helpful because it lets us organize things better. It keeps the rules for how things look away from the rules for what's on the page. This makes it easier to use the same look for different parts of a webpage.

CSS SYNTAX

  • Picking Things:CSS uses selectors to pick things on a webpage. These can be the names of things, special names, or how things relate to each other.

  • Look and Feel: CSS controls how things look. You can change colors, sizes, spaces, and more to make things look nice.

  • Instructions:Each thing you pick gets some instructions on how it should look. For example, you can say, "Make this red" or "Make this bigger."

  • Brackets:The instructions are put inside curly brackets {}. It's like putting the rules in a box.

  • Talking to Many Things: You can talk to many things at once and give them the same instructions. For example, if you want all titles to be blue, you say, "Make all titles blue."

  • Hidden Notes: You can leave notes for yourself using /* */. These notes won't show on the webpage, but you can write helpful things for yourself.

  • Order Matters: Sometimes, the way you say things matters. If you say something one way and then change it another way, the last way is what the webpage will do.

  • Sharing Styles: Sometimes, if something looks a certain way, the things inside it will also look the same. It's like if a big box is blue, the small things inside it might also be blue.

  • Types of Numbers: When you say how big or small something is, you can use different types of numbers. Like saying, "Make this 10 big" or "Make this half as big."

Understanding these simple points can help you work with CSS to make things on your website look presentable, neat, and attractive

Examples:

  • Picking Things: To select all paragraphs in CSS, you use the selector p. For example:
    p { color: #333; font-size: 16px; }
    This will make all paragraphs have a color of dark gray and a font size of 16 pixels.
  • Look and Feel: Changing the background color of a webpage's body:
    body { background-color: #f7f7f7; /* Light gray background */ }
  • Instructions: Making a specific paragraph red:
    .red-paragraph { color: red; }
    Apply this class to a paragraph like this: <p class="red-paragraph">This is a red paragraph</p>.
  • Brackets: Wrapping instructions in brackets:
    h1 { font-size: 24px; }
  • Talking to Many Things: Making all headings blue:
    h1, h2, h3, h4, h5, h6 { color: blue; }
  • Hidden Notes: Leaving a note for yourself:
    /* This is a comment */ p { line-height: 1.5; }
  • Order Matters: Setting different styles for links:
    a { color: blue; /* Default link color */ }
    a:hover { color: red; /* Link color on hover */ }
  • Sharing Styles: Making all elements inside a container have the same font:
    .container { font-family: 'Arial', sans-serif; }
  • Types of Numbers: Specifying different sizes for an element:
    .large-text { font-size: 24px; }
    .small-text { font-size: 14px; }
    Apply these classes to elements to control their text size.
